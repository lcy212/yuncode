반복문

3가지 이상으 기능/ 요소 등의 처리를 반복 수행할 경우에는 반복문을 사용하여 처리

  • while : 여러개의 반복수행 처리. 처음부터 수행하지 않을 수 있음. while(){} 다른 반복문보다 속도 느림, 조건 수행 후 반복끝내는 기능을 미작성하면 마비됨
  • do-while : while 확장형. 조건 수행 시 반복문 무조건 최소 1번 수행. do {} while()
  • for : 가장 많이 사용하는 반복문. 최초값, 조건, 수행에 대한 내용을 모두 작성하여 수행 속도를 개선함 for( ; ; ){}
  • for-in : for 변형 형태. 배열보다는 객체에서 많이 사용함. for(in){}
  • forEach : for 변형 형태. 배열의 반복으로 사용되며 배열요소를 각각 수행한다는 의미 [].forEach(function(data, index ){})
