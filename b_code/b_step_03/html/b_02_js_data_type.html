데이터 타입 (형타입)

기본형

  • 숫자 : js에서는 실수, 정수 구분 없이 숫자
  • 문자 : 반드시 '' 내부에 작성하는 형태. 숫자더라도 '' 내부에 있으면 문자
  • 논리형 : 참(true), 거짓(false)을 의미하는 결과물을 도출 - true, false는 문자 아님
  • undefined : 값이 지정되지 않은 상태. 변수나 그 무엇은 존재/유지
  • null : 값이 없거나, 값 전달시 없는 값

특수형, 복합형

  • 객체
    • 배열 : html문서에서 말하는 ol 처럼 여러개의 목록으로 []안에 내용을 나열하는 것
    • 객체 : 배열처럼 나열하지만 순서 없음. 해당 목록에 대한 정의를 하는 것으로 html의 dl, css의 속성의 개념과 같다. {}내부에 여러개의 내용을 담아 그룹화 시킨다.
  • 함수 : 행동처리를 위해 만든 묶음(바로실행, 내용담는 기능X)
    (선언식, 표현식, 즉시실행함수)

다른 프로그래밍 언어와 다른점 : 객체 내부에 배열, 함수를 담을 수 있다.
