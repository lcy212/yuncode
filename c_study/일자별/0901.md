# 첫 수업

## 네이밍

### 네이밍 방법
- directory / folder
    - 특수문자를 사용하지않는다. 단, -와 _은 가능하다.
    - 첫 글자는 영문 소문자를 사용한다.
    - 네이밍 중간에 숫자는 허용된다. 날짜를 작성할 때 유용하다.

### 네이밍 기법
1. 카멜케이스 : 중간에 대문자를 사용한다. 예시) wordWord<br />
               대부분의 컴퓨터언어에서 사용한다.
2. 언더스코어 : 스네이크 케이스라고도 한다. 예시) word_word
3. 웹 언어의 경우 : 예시) word-word
