## style

- background-clip : 배경색을 칠하는 범위
  - content-box : 내용만
  - padding-box : padding 범위까지
  - border-box : 외곽선까지

- calc는 구형 브라우저에서 지원안함

  따라서 calc를 쓰는 요소는 두 번 적어두는 편

  ex) width:980px; width:calc(100% - 320px);

- 페이지 px이 커지는 것을 고려하여 width를 100%로 작성하는 것 고민해보기
